enum pep_error_t

PEP client error codes.

Use pep_strerror(pep_error_t errno) to retrieve the error description.

See also:
pep_strerror(pep_error_t errno)
Enumerator:
PEP_OK 

OK, No error.

PEP_ERR_MEMORY 

Memory allocation error.

NULL pointer exception.
