Argus PEP client C API

The Argus PEP client API for C is a multi-thread safe client library used to communicate with the Argus PEP Server. It authorizes request and receives authorization response back from Argus.

Runtime dependencies

  • libcurl package: curl >= 7.15

Documentation

Please refer to https://twiki.cern.ch/twiki/bin/view/EGEE/AuthorizationFramework wiki for more information.
